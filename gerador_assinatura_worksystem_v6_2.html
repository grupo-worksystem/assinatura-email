HTML para Outlook
Como usar no Outlook:
  1. Clique em Copiar HTML
  2. Outlook → Arquivo → Opções → Email → Assinaturas
  3. Crie ou edite → clique no ícone HTML e cole o código
  4. Salve e defina como padrão
